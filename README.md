# SQL Project

Простой инструмент для визуализации структуры SQLite-базы данных и работы с её содержимым.

## О проекте

Это приложение открывает SQLite-файл, строит ER-диаграмму таблиц и внешних ключей, отображает данные таблиц
и позволяет выполнять заранее подготовленные SQL-запросы.

Основные возможности:
- загрузка SQLite-базы и построение графа таблиц/связей;
- автоматическая и ручная расстановка таблиц на диаграмме;
- сохранение и восстановление положения таблиц в файле `.layout.json`;
- миникарта для навигации по диаграмме;
- просмотр данных выбранной таблицы;
- выполнение готовых SQL-шаблонов с параметрами;
- экспорт диаграммы в PNG и SVG.

## Структура проекта

- `run_test_view.py` — основной запуск приложения в текущей версии.
- `src/main.py` — альтернативная точка входа для запуска через модуль.
- `src/controller/app_controller.py` — логика открытия БД, рендера диаграммы, обмена данными с UI и сохранения макета.
- `src/model/db_parser.py` — парсер SQLite-схемы, извлекает таблицы, поля и внешние ключи.
- `src/model/graph_model.py` — хранит модель таблиц и связей, сохраняет/загружает позиции элементов.
- `src/view/` — визуальное представление:
  - `main_window.py` — главное окно с вкладками «Диаграмма», «Данные» и «Запросы»;
  - `diagram_scene.py` — QGraphicsScene, создаёт таблицы и связи;
  - `diagram_view.py` — вид, поддерживает масштабирование и вписание в окно;
  - `items/table_item.py` — графический элемент таблицы;
  - `items/edge_item.py` — графический элемент связи;
  - `widgets/minimap.py` — миникарта для быстрой навигации.
- `src/utils/layout_engine.py` — алгоритм автоматической раскладки диаграммы.
- `src/utils/routing_engine.py` — базовая логика прокладки связей.

## Установка

1. Создайте и активируйте виртуальное окружение:

```powershell
python -m venv venv
venv\Scripts\Activate.ps1
```

2. Установите зависимости:

```powershell
pip install -r requirements.txt
```

## Запуск

```powershell
python run_test_view.py
```

или

```powershell
python -m src.main
```

## Как пользоваться

1. Откройте SQLite-файл через меню `Файл → Открыть БД...`.
2. Диаграмма отобразит таблицы и связи по внешним ключам.
3. Вкладка `Данные` позволяет выбрать таблицу и посмотреть первые строки.
4. Вкладка `Запросы` предлагает готовые SQL-шаблоны с заполнением полей.
5. В меню `Вид` доступно масштабирование, вписать в окно и авто-раскладка.
6. Сохранение макета сохраняет текущие позиции таблиц в JSON-файл рядом с БД.

## Цель проекта

Проект создан как рабочая визуальная утилита для анализа структуры SQLite-базы, изучения её таблиц и связей,
а также для быстрого просмотра данных и выполнения типовых SQL-запросов.
